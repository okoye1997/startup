# CodeTrack Landing Page

A responsive, static landing page for **CodeTrack** — a concept app for building a daily coding habit and tracking learning streaks.

## Preview

Open `index.html` directly in a browser, or serve the `codetrack` folder with a local development server such as VS Code Live Server.

No installation, build step, or environment variables are required.

## Included sections

- Header navigation and calls to action
- Hero section with a coding-activity preview
- Community statistics and avatar group
- Product feature cards and three-step workflow
- Customer testimonial with the Maya attribution
- Heatmap-style final call to action and footer navigation

## Project structure

```text
codetrack/
|-- images/       # Logo and page images
|-- notes/        # Class notes and design-system documentation
|-- index.html    # Page content and structure
|-- style.css     # Visual styles and responsive rules

`-- README.md
```

## Technologies

- HTML5
- CSS3
- Google Fonts: Space Grotesk, Inter, and JetBrains Mono
- Boxicons (mobile menu icon)

## Responsive design

The page adapts from large desktop screens down to small phones. Its main responsive adjustments cover navigation, the two-column hero, calls to action, statistics, cards, testimonial, heatmap, and footer.

## External resources

The page loads its fonts from Google Fonts and the mobile menu icon from the Boxicons CDN. An internet connection is needed for those external resources to load; the page content and local images are included in this project.

## Design reference

The visual design uses a green CodeTrack palette, rounded cards, and a contribution-style heatmap. See [notes/designsystem.md](notes/designsystem.md) for the design reference.
